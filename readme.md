# 🍽️ Flavour Finds

A modern food blogging website built using **Node.js**, **Express.js**, **EJS**, and **CSS**.

Flavour Finds allows users to:

* Create blogs
* Edit blogs
* Delete blogs
* Choose blog categories
* Display random food images
* Show author names dynamically

---

# ✨ Features

## 📝 Blog Management

* Create new blogs
* Edit existing blogs
* Delete blogs
* Dynamic rendering using EJS

## 🍔 Food Categories

Users can publish blogs under categories like:

* Recipes
* Street Food
* Desserts
* Healthy Food
* Fast Food

## 🖼️ Random Food Images

Every blog is displayed with a randomly selected food illustration/image.

## 👤 Dynamic User Greeting

The website greets users dynamically after entering their name.

## 📱 Responsive Design

Responsive UI built using pure CSS.

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/flavour-finds-blog.git
```

---

## 2️⃣ Open Project Folder

```bash
cd flavour-finds-blog
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Run Server

```bash
nodemon index.js
```

or

```bash
node index.js
```

---

# 🌐 Open In Browser

Visit:

```bash
http://localhost:3000
```
